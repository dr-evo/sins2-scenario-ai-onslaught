Custom scenario where multiple players fight powerful AI.

Modifies simulation - all players in multiplayer must have this mod.

Note: This mod changes behavior of Impossible AI for all scenarios while active 

Built for patch 1.45 (Eivonn's fine) , 2025-10-06+



# Features
- scenario "AI Onslaught", 3 AI vs 6 players
  - AI starts with solid planets and unlocked techs up to tier 2
- Adjusted impossible difficulty ("Onslaught AI")
  - double supply cap
  - significantly increased base income
  - faster exotic build time
  - more civilian/military slots
  - faster ship build time
  - increased capital ship xp gained
  - gives less xp per kill to players
  - builds strike craft faster
  - culture resistance
  - does not build super weapons
